# wordbook-gnsl

ngsl,nawl,tsl,bslの単語帳。

# 使い方
1. `db_command.py/init_db()`
1. `db_command.py/insert_new_version()`
1. `python3 main.py`

## insert_new_version
### 引数
#### name
Versionに付ける名前。

### category
作成するVersionの対象テスト範囲
'ngsl', 'nawl', 'tsl', 'bsl'のいずれかの文字列を渡すことでテスト範囲を指定する。
